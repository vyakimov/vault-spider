# Larder

The pantry/freezer tracker — a phone-first web app on SQLite, built to stop buying a fourth jar
of cumin. It runs on [[PuddleJumper]] and is reachable only over the tailnet; see
[[LAN-only larder.foxglove.example]].

## Plans
- [[Plan — Flask HTMX]] — the one that was built: Flask + HTMX on SQLite, barcode input from the
  phone camera.
- [[Plan — Datasette and SQLite]] — the simpler alternative considered first.
