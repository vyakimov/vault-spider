# A list of all tools
A [[Marionette]] note.



**File Operations**

- `read` — Read file contents
- `write` — Create or overwrite files inside the workspace

**Execution**

- `shell` — Run allowlisted shell commands (gateway or node)
- `process` — Manage background shell sessions

**Web**

- `web_search` — Search the web
- `web_fetch` — Fetch and extract readable content from a URL

**Scheduling**

- `cron` — Manage cron jobs and wake events

**Messaging**

- `message` — Send Signal messages and channel actions

**Media**

- `image` — Analyze an image with the configured image model

This list is defined by the gateway's tool policy for this agent. See also [[Cron vs Wake Events]].
