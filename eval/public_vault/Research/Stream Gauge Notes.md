# Stream Gauge Notes

## Setup

A staff gauge bolted to the footbridge pier, read manually on field visits, photographed each
time so readings can be audited later. No telemetry: the stream is a context variable, not a
primary instrument.

## Rating curve

Six manual flow measurements so far, all at low or moderate stage. The fitted curve is not to
be trusted above the highest measured stage; high-flow readings are recorded as stage only and
flagged, never converted to discharge.

## Winter

The gauge board ices over. Winter readings are opportunistic and marked as approximate.
